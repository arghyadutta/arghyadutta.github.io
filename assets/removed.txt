    <h3>Addons for Mozilla Firefox</h3>
    <ol>
        <li><a href="https://addons.mozilla.org/en-US/firefox/addon/ublock-origin/">uBlock Origin</a> is an essential ad blocker. If you're adventurous, it even lets you disable all javascript on a website and remove ad banners or similar elements.</li>
        <li><a href="https://addons.mozilla.org/en-US/firefox/addon/privacy-badger17/">Privacy Badger</a> is not an ad-blocker, but it does something <a href="https://privacybadger.org/#What-is-Privacy-Badger">very clever</a>.</li>
        <li>The <a href="https://www.zotero.org/download/connectors">Zotero connector</a> allows you to add papers to your Zotero Library. It can also automatically redirect journal webpages through an institutional proxy, allowing you to download papers with your institutional credentials.</li>
        <li><a href="https://addons.mozilla.org/en-US/firefox/addon/return-youtube-dislikes/">Return YouTube dislikes</a> brings back a useful feature.</li>
        <li><a href="https://addons.mozilla.org/en-US/firefox/addon/single-file/">Single-file</a> is a neat add-on that can download a webpage with everything, including figures, in a <em>single</em> HTML file. Very useful if you want to locally archive, for example, a blog post or a Twitter thread.</li>
        <li><a href="https://addons.mozilla.org/en-US/firefox/addon/tabliss/">Tabliss</a> sets the new tab page.</li>
        <li><a href="https://addons.mozilla.org/en-US/firefox/addon/tranquility-1/">Tranquility Reader</a> converts any webpage with an article to a simple one with only the article&#39;s text and core images so that you can read it without 1000 flashy things annoying you.</li>
        <li><a href="https://addons.mozilla.org/en-US/firefox/addon/feedbroreader/">Feedbro</a> helps to get articles from any website that provides an RSS feed (blogs, journals, magazines…). A good alternative to Feedly.</li>
        <li><a href="https://addons.mozilla.org/en-US/firefox/addon/gesturefy/">Gesturefy</a> is such a neat add-on! It lets you do several things, like going back to the previous webpage, only with gestures of your mouse.</li>
        <li>You open a webpage like <a href="http://bactra.org">Cosma Shalizi&#39;s</a> (check it!) and find hundreds of links. Which one is active and which one is dead? <a href="https://addons.mozilla.org/en-US/firefox/addon/link-analyzer/">Link analyzer</a> does that job for you with one click.</li>
        <li><a href="https://addons.mozilla.org/en-US/firefox/addon/hide-youtube-shorts/">Hide Youtube-Shorts</a> removes YouTube shorts from its homepage, subscriptions page, and search results. It can also hide the &quot;Shorts&quot; tab. Good riddance!</li>
    </ol>
