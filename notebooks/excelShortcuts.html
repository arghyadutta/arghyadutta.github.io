Excel keyboard shortcuts for Mac

Why you may not want to use this: It's ChatGPT-generated content!

I am not a fan of Excel, but also can't avoid it at work. So, I made this cheat-sheet of sorts using ChatGPT. I haven't tested most of them. I plan to use and edit it as needed. (Last updated: 2025-08-25, Monday)

Editing & Entering

Shortcut Action
Control + U Edit the active cell without overwriting
Type directly Replace cell contents
Return Enter and move down
Shift + Return Enter and move up
Tab Enter and move right
Shift + Tab Enter and move left
Control + Return Enter without moving
Escape Cancel editing
Control + Option + Return Insert line break inside a cell (sometimes Option + Command + Return)

Navigation

Shortcut Action
Arrow keys Move one cell
Control + Arrow Jump to edge of current data region
Command + Arrow Jump to beginning/end of row or column
Fn + Left / Right Home / End
Fn + Up / Down Page Up / Page Down
Control + G or F5 Go To (jump to cell)
Command + [ Navigate to precedent cell
Command + ] Navigate to dependent cell

Selection

Shortcut Action
Shift + Arrow Extend selection by one cell
Control + Shift + Arrow Select to edge of data region
Command + A Select current region; press again to select entire sheet
Shift + Space Select entire row
Command + Space Select entire column
Command + Shift + Space Select entire worksheet

Editing Aids

Shortcut Action
Command + Z Undo
Command + Y or Shift + Command + Z Redo
Command + C Copy
Command + X Cut
Command + V Paste
Command + Option + V Paste Special
Delete Clear contents

Formatting

Shortcut Action
Command + 1 Format cells dialog
Command + B Bold
Command + I Italic
Command + U Underline

Other Useful Shortcuts

Shortcut Action
Command + T Create table
Shift + F11 Insert new worksheet
Command + Option + R Recalculate all formulas
Command + Arrow + Shift Select to beginning/end of row/column
Command + Tab Switch applications

Function Key Shortcuts

Shortcut Action Notes
F2 Edit active cell Often overridden by macOS; use Control + U in Excel instead
F4 Repeat last action May require Fn + F4 depending on keyboard settings
Shift + F2 Edit cell comment May require Fn
F9 Calculate selected cells May require Fn
Shift + F9 Calculate active worksheet May require Fn
Command + = Insert AutoSum No conflict

macOS vs Excel Conflicts

Shortcut macOS Action Excel Action Notes
Command + Space Spotlight search Select entire column You must disable/reassign Spotlight in System Settings
Control + Space Input source change (language) Select entire column in some Excel versions Reassign in macOS Keyboard Shortcuts
Fn + F-keys macOS features (brightness, volume) Excel function keys Change in System Settings → Keyboard: “Use F1, F2, etc. keys as standard function keys”
Command + Tab Switch applications Not used by Excel macOS always overrides
